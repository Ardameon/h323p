## 📌 Статус проекта

- **Фаза:** Проектирование завершено (100%)

- **Последнее обновление:** 2026-03-16

## 📖 Описание

**h323p** — утилита для тестирования H.323 вызовов по аналогии с SIPP для SIP.

### Цели
- Нагрузочное тестирование endpoints/gateways/gatekeepers
- Функциональное тестирование протокола
- Fuzzing тестирование
- Поддержка аудио, видео, H.239

### Технические решения
| Решение | Выбор |
|---------|-------|
| **Язык** | C++17 |
| **H.323 стек** | H323Plus + PTLib |
| **Сборка** | CMake |
| **Сценарии** | XML (SIPP-подобный) |
| **Тесты** | CppUTest |

## 🚀 Быстрые команды

```bash
# Сборка
mkdir build && cd build
cmake .. -DH323PLUS_DIR=/path/to/h323plus
make -j4

# Запуск
./h323p call 192.168.1.100          # Вызов
./h323p load 192.168.1.100 -c 100   # Нагрузка
./h323p listen                      # Прослушивание
./h323p fuzz 192.168.1.100 -T rtp   # Fuzzing
./h323p info --version              # Информация

# Тесты
ctest --output-on-failure
```
